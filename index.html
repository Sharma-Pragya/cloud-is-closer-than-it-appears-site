Cloud Is Closer Than It Appears

Pragya Sharma, Hang Qiu, Mani B. Srivastava
UCLA & UC Riverside
ICCCN 2025
Paper Code Slides

Abstract

We revisit the assumption that cloud-based inference is unsuitable for real-time control tasks. Through analytical modeling and empirical validation using autonomous braking scenarios in CARLA, we demonstrate that cloud inference can outperform local inference under common CPS conditions. Our framework identifies platform-performance regimes where cloud is not only viable—but preferable.
